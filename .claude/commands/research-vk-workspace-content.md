# Поиск тем и форматов контента VK WorkSpace

Команда для комплексного исследования контент-возможностей VK WorkSpace.

## Использование

```bash
/research-vk-workspace-content
```

Или с параметрами:

```bash
/research-vk-workspace-content [scope]
```

Где scope:
- `полный` — Полный аудит: конкуренты + рынок + тренды + инфоповоды
- `конкуренты` — Фокус на контенте конкурентов
- `тренды` — Актуальные тренды и инфоповоды
- `точечный` — Темы под конкретный канал или продукт

## Что делает команда

Запускает агента **vk-workspace-content-researcher** со скиллами:
- `vk-workspace-content-research` — методология исследования, карта конкурентов, модель VISTA
- `vk-workspace-knowledge` — 13 продуктов VK WorkSpace, аудитории, позиционирование

## Процесс

### 1. Определение scope
- Масштаб исследования (полный / конкуренты / тренды / точечный)
- Продуктовый фокус (какие сервисы VK WorkSpace)
- Каналы размещения (блог, VC.ru, колонки в СМИ)
- Целевая аудитория (опционально)

### 2. Исследование
- **Конкуренты:** Яндекс 360, Битрикс24, МойОфис, Труконф, Kaiten и др.
- **Рынок:** Состояние рынка корпоративных коммуникаций в России
- **Тренды:** AI, импортозамещение, безопасность, суперапп и др.
- **Инфоповоды:** Продуктовые, рыночные, регуляторные, событийные

### 3. Скоринг по модели VISTA
Каждая контент-возможность оценивается по 5 критериям:
- **V**alue — Ценность для аудитории (x2)
- **I**mpact — Охват и вовлечение (x2)
- **S**tanding — Позиционирование VK WorkSpace (x1.5)
- **T**iming — Тайминг и актуальность (x1.5)
- **A**dvantage — Конкурентное преимущество (x1)

Максимум: 85 баллов

### 4. Контент-план
- Приоритизированная матрица: тема × формат × канал
- Детальные карточки для топ-10 рекомендаций
- Календарь публикаций

## Результат

Отчёт сохраняется в: `content-hub/research/drafts/{YYYY-MM-DD}-vk-workspace-content-plan.md`

### Структура отчёта:
1. **Executive Summary** — ключевые находки и рекомендации
2. **Конкурентная среда** — что публикуют конкуренты, где пробелы
3. **Рыночные тренды** — актуальные тренды с оценкой релевантности
4. **Инфоповоды** — актуальные и прогнозируемые инфоповоды
5. **Контент-план по каналам** — таблицы для блога, VC.ru, колонок
6. **Топ-10 рекомендаций** — детальные карточки с обоснованием
7. **Календарь публикаций** — план по неделям

## Примеры использования

### Квартальное планирование
```bash
/research-vk-workspace-content полный
```
Результат: комплексный контент-план на квартал с 15-20 темами по всем каналам.

### Оперативный поиск тем
```bash
/research-vk-workspace-content тренды
```
Результат: 5-10 тем, привязанных к актуальным инфоповодам и трендам.

### Анализ конкурентов
```bash
/research-vk-workspace-content конкуренты
```
Результат: карта контента конкурентов + пробелы + рекомендации по дифференциации.
