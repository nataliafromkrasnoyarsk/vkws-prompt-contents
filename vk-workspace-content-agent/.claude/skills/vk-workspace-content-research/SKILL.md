## 2. Каналы размещения контента VK WorkSpace

### 2.1. Блог workspace.vk.ru

| Параметр | Значение |
|----------|----------|
| **Тип контента** | SEO-статьи для органического трафика |
| **Объём** | 1500-3000 слов |
| **Форматы** | Обзор-сравнение, Гайд-инструкция, Образовательная, Продуктовый обзор |
| **Упоминания VK WorkSpace** | 3-5 раз, органично через сценарии |
| **Тон** | Профессиональный, консультативный, доступный |
| **SEO** | Primary + secondary KW, meta title/description, FAQ, H2/H3 с ключами |
| **CTA** | "Попробуйте VK WorkSpace бесплатно 30 дней" |
| **Аудитория** | ИТ-директора, сисадмины, SMB-руководители через поисковый трафик |
| **Агент для создания** | `/seo-article-vk-workspace` |

**Лучшие темы для блога:**
- "Чем заменить [X]" — сравнительные обзоры (высокий поисковый спрос)

## 3. Модель VISTA для скоринга контент-возможностей

### 3.1. Описание критериев

#### V — Value (Ценность для аудитории) [вес x2]

| Балл | Описание |
|------|----------|
| 9-10 | Решает острую боль аудитории, даёт уникальные данные или пошаговую инструкцию |
| 7-8 | Полезная информация, практические выводы, экономит время читателя |
| 5-6 | Информативно, но без уникальных инсайтов — "ещё одна статья о..." |
| 3-4 | Поверхностно, мало практической пользы |
| 1-2 | Чистый промо или пересказ очевидного |

**Маркеры высокой ценности:**
- Уникальные данные (исследования VK Tech, внутренняя статистика)
- Пошаговые инструкции с чеклистами
- Реальные кейсы с цифрами (до/после)
- Экспертная оценка тренда (не пересказ новости)
- Сравнительные таблицы с критериями

#### I — Impact (Охват и вовлечение) [вес x2]

| Балл | Описание |
|------|----------|
| 9-10 | Вирусный потенциал: провокация, шокирующие данные, массовый интерес. 10000+ просмотров |
| 7-8 | Устойчивое вовлечение: тема в фокусе аудитории, SEO-потенциал. 5000-10000 просмотров |
| 5-6 | Нишевая тема, ограниченная аудитория, но лояльная. 2000-5000 просмотров |
| 3-4 | Узкоспециализированная тема. 500-2000 просмотров |
| 1-2 | Минимальный интерес. <500 просмотров |

**Факторы высокого охвата:**
- Тема затрагивает массовую аудиторию (импортозамещение, удалёнка, продуктивность)
- Провокационный или контринтуитивный угол
- Привязка к актуальному инфоповоду
- SEO-потенциал (высокий поисковый спрос)
- Формат с высоким вовлечением (кейс, мнение, сравнение)

#### S — Standing (Позиционирование VK WorkSpace) [вес x1.5]

| Балл | Описание |
|------|----------|
| 9-10 | Напрямую усиливает ключевые RTB: экосистемность, безопасность, российский вендор |
| 7-8 | Демонстрирует экспертизу VK WorkSpace в конкретной области |
| 5-6 | Косвенно связано с позиционированием, тема в периметре бренда |
| 3-4 | Слабая связь с брендом, но не противоречит |
| 1-2 | Не связано с позиционированием VK WorkSpace |

**RTB VK WorkSpace (из скилла vk-workspace-knowledge):**
1. Удобно — все сервисы в одном приложении, сквозные сценарии
2. Надёжно — российский разработчик, крупнейшие компании доверяют
3. Для бизнесов любого размера — SaaS, On-Premises, Hosted, ПАК
4. Безопасность — корпоративный уровень защиты

#### T — Timing (Тайминг и актуальность) [вес x1.5]

| Балл | Описание |
|------|----------|
| 9-10 | Горячий инфоповод (1-2 недели), публикация создаст резонанс |
| 7-8 | Актуальный тренд, тема "на хайпе" в индустрии |
| 5-6 | Тема актуальна, но не привязана к конкретному моменту |
| 3-4 | Вечнозелёная тема, может подождать |
| 1-2 | Тема устарела или не привязана к контексту |

**Календарь тайминга:**
- **Январь-Февраль:** Планы на год, бюджеты, стратегии цифровизации
- **Март-Апрель:** Реализация проектов, весенние конференции
- **Май-Июнь:** Конференции (Highload++, DevOpsConf), итоги полугодия
- **Июль-Август:** Вечнозелёный контент, гайды, образовательный контент
- **Сентябрь-Октябрь:** Осенние конференции, возврат к планированию
- **Ноябрь-Декабрь:** Бюджетирование, тендеры, итоги года, прогнозы

#### A — Advantage (Конкурентное преимущество) [вес x1]

| Балл | Описание |
|------|----------|
| 9-10 | Конкуренты НЕ писали, уникальные данные/опыт, первый на рынке |
| 7-8 | Конкуренты писали слабо, есть возможность сделать значительно лучше |
| 5-6 | Конкуренты покрыли тему, но можно добавить уникальный угол |
| 3-4 | Тема хорошо покрыта конкурентами, дифференциация слабая |
| 1-2 | "Me too" контент без дифференциации |

**Источники конкурентного преимущества VK WorkSpace:**
- Уникальные данные исследования VK Tech (750 компаний)
- Экосистемный подход (13 сервисов в одном приложении)
- 4 модели развёртывания (ни у кого больше нет)
- Реальные спикеры с опытом внедрений
- Кейсы крупных российских компаний

### 3.2. Формула и пороги

```
VISTA Score = V×2 + I×2 + S×1.5 + T×1.5 + A×1

Максимум: 10×2 + 10×2 + 10×1.5 + 10×1.5 + 10×1 = 85
```

| Диапазон | Приоритет | Действие |
|----------|-----------|----------|
| **70-85** | Must-do | Немедленная реализация. Назначить автора и дедлайн |
| **55-69** | Should-do | Включить в план на ближайший месяц |
| **40-54** | Could-do | Резерв. Реализовать при наличии ресурсов |
| **< 40** | Won't-do | Отложить или отбросить |

## 4. Актуальные тематические кластеры

### Кластер 1: Импортозамещение корпоративных коммуникаций
**Ключевые темы:**
- Замена Microsoft 365 / Google Workspace полным стеком
- Замена Teams / Slack корпоративным мессенджером
- Замена Zoom / Webex российскими ВКС
- Замена Jira → Проекты VK WorkSpace
- Замена Miro → Доска VK WorkSpace
- Итоги и уроки 3 лет импортозамещения
- Реестр ПО: что изменилось, что ожидать

**SEO-потенциал:** Высокий (поисковый спрос на "замена [продукт]", "аналог [продукт]")
**VC.ru потенциал:** Высокий (кейсы миграции с цифрами)
**Колонки:** Высокий (стратегический взгляд на импортозамещение)

### Кластер 2: Экосистемность и суперапп
**Ключевые темы:**
- Цифровая фрагментация: ~1200 переключений между приложениями в день
- Суперапп vs набор разрозненных инструментов
- Кросс-сценарии: непрерывность рабочего процесса
- Единая платформа = экономия TCO
- 34% компаний выбирают одного поставщика

**SEO-потенциал:** Средний (нишевые запросы)
**VC.ru потенциал:** Высокий (мнения о подходе, провокационные тезисы)
**Колонки:** Высокий (стратегическое позиционирование)

### Кластер 3: Безопасность корпоративных коммуникаций
**Ключевые темы:**
- Telegram для работы: риски и альтернативы
- 77% пользователей мессенджеров сталкивались с мошенничеством
- 355 000 DDoS-атак за полугодие — коммуникации как цель
- ФЗ-152, ФСТЭК, ГОСТ Р 57580 — compliance как бизнес-требование
- DLP-интеграция: InfoWatch, Solar Dozor, SearchInform
- MAM: управление мобильными приложениями
- Zero trust для корпоративных коммуникаций

**SEO-потенциал:** Высокий (запросы про безопасность, compliance)
**VC.ru потенциал:** Высокий (угрозы, утечки, провокации про Telegram)
**Колонки:** Высокий (экспертиза по ИБ, регуляторика)

### Кластер 4: Удалённая и гибридная работа
**Ключевые темы:**
- Гибридный формат: как организовать коммуникации
- Mobile-also: мобильный опыт для распределённых команд
- ВКС: от "замены Zoom" к полноценной платформе встреч
- Оффлайн-режим: работа без интернета
- Кроссплатформенность: Windows, macOS, Linux, iOS, Android, Huawei

**SEO-потенциал:** Средний (конкурентные запросы)
**VC.ru потенциал:** Средний (тема "отстрелялась", но есть новые углы)

- Замена Jira: что может Проекты VK WorkSpace

## 5. Форматы контента: матрица канал × формат

### 5.1. Какие форматы работают на каждом канале

| Формат | Блог (SEO) | VC.ru | Колонки в СМИ |
|--------|-----------|-------|---------------|
| **Обзор-сравнение** | +++ (SEO-хит) | ++ (если с баллами) | + (для CNews, TAdviser) |
| **Гайд-инструкция** | +++ (длинный хвост) | ++ (с чеклистом) | - |
| **Образовательная статья** | +++ (вход в воронку) | + (если с данными) | + (для Habr) |
| **Продуктовый обзор** | ++ (для блога) | + (только через кейс) | - |
| **Кейс с цифрами** | ++ | +++ (топ-формат) | ++ (для CNews, TAdviser) |
| **Мнение / opinion** | - | +++ (комментарии!) | +++ (профильный формат) |
| **Тренд-разбор** | ++ | +++ (с данными) | +++ (для Forbes, РБК) |
| **Честный разбор / факап** | - | +++ (вирусный) | + (для VC.ru-колонки) |
| **Разбор новости** | + | +++ (скорость!) | ++ (комментарий эксперта) |
| **Дайджест / рейтинг** | ++ | ++ | + |
| **Сравнение / поединок** | +++ (SEO) | ++ (если объективно) | - |

Легенда: +++ идеально, ++ подходит, + возможно, - не рекомендуется

## 8. Чеклист качества исследования

### Перед выдачей результатов проверь:

**Источники:**
- [ ] Использованы данные из WebSearch (не только база знаний)
- [ ] Конкурентный контент проанализирован на реальных площадках
- [ ] Минимум 3 источника для каждого тренда
- [ ] Данные актуальны (не старше 3-6 месяцев для трендов)

**Рекомендации:**
- [ ] Каждая тема привязана к конкретному каналу
- [ ] Каждая тема имеет конкретный угол (не "напиши о чём-нибудь")
- [ ] Каждая тема привязана к продукту VK WorkSpace
- [ ] VISTA-скоринг проведён для каждой рекомендации
- [ ] Указан источник идеи (конкурентный пробел / тренд / инфоповод)

**Баланс:**
- [ ] Есть и "быстрые" (реактивные на инфоповоды) и "стратегические" темы
- [ ] Покрыты все выбранные каналы
- [ ] Разнообразие форматов (не только один тип)
- [ ] Учтена сезонность и тайминг

**Практичность:**
- [ ] Для колонок указан спикер
- [ ] Для SEO-статей указаны потенциальные ключевые слова
- [ ] Для VC.ru указан формат и hook
- [ ] Результат можно передать напрямую агентам создания контента

## 9. Red Flags

### Плохие рекомендации (избегать):
- Абстрактные темы без угла ("напишите о безопасности")
- Темы без привязки к каналу
- "Me too" контент, который уже есть у конкурентов
- Темы без proof points (нет данных, кейсов, экспертизы)
- Промо-контент, замаскированный под экспертизу
- Устаревшие тренды, выдаваемые за актуальные
- Темы, в которых VK WorkSpace не имеет конкурентного преимущества
- Игнорирование требований конкретного канала (Forbes-формат на VC.ru)

### Хорошие рекомендации (стремиться):
- Конкретная тема + чёткий угол + формат + канал + аудитория + продукт
- Основана на реальном пробеле в контенте конкурентов
- Привязана к актуальному тренду или инфоповоду
- Усиливает RTB VK WorkSpace через экспертизу, а не промо
- Имеет высокий VISTA Score с обоснованием
- Можно сразу передать агенту создания контента для реализации
