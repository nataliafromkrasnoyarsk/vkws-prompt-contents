# /event-page-wizard:create-event

Запускает пошаговый wizard для создания страницы мероприятия.

## Usage

```
/event-page-wizard:create-event
```

Или просто напишите:
- "Создай страницу вебинара"
- "Страница конференции"
- "Создать ивент"

## Event Types

| Тип | Описание | Формат по умолчанию |
|-----|----------|---------------------|
| вебинар | Онлайн-трансляция | online |
| конференция | Масштабное мероприятие | hybrid |
| митап | Неформальная встреча | offline |
| воркшоп | Практическое занятие | offline |
| партнёрское | С партнёром | varies |

## Navigation Commands

| Команда | Действие |
|---------|----------|
| `назад` | Вернуться на шаг назад |
| `пропустить` | Пропустить опциональный шаг |
| `статус` | Показать текущие данные |
| `начать заново` | Перезапустить wizard |
| `помощь` | Показать справку |

## Output Formats

После завершения wizard генерирует:

### 1. Markdown-страница
Готовая страница для сайта в формате Markdown.

### 2. JSON-структура
```json
{
  "event": {
    "type": "вебинар",
    "title": "Название",
    "date": "2024-10-22",
    "time": "11:00"
  },
  "speakers": [...],
  "program": [...],
  "partners": [...]
}
```

## Quick Start Options

### С предзаданным типом
```
/event-page-wizard:create-event --type вебинар
```

### Быстрый режим (минимум вопросов)
```
/event-page-wizard:create-event --quick
```

## Examples

### Простой вебинар
```
/event-page-wizard:create-event --type вебинар --quick
```

### Конференция с полной программой
```
/event-page-wizard:create-event --type конференция
```
