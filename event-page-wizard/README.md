# Event Page Wizard Plugin

Интерактивный агент для создания страниц мероприятий в стиле VK Cloud.

**Версия:** 1.1.0

## Что делает плагин

Проводит пользователя через **10-шаговый wizard**, собирая данные для генерации страницы мероприятия (вебинара, конференции, митапа, хакатона).

**Результат:** готовая Markdown-страница + JSON-структура, сохранённые в файлы.

## Быстрый старт

### Установка

Скопируйте папку `event-page-wizard` в директорию вашего проекта или в `~/.claude/plugins/`.

### Запуск

```bash
/event-page-wizard:create-event
```

Или с параметрами:
```bash
/event-page-wizard:create-event --type вебинар --quick
```

Или просто напишите:
```
Создай страницу вебинара
```

## Параметры запуска

| Флаг | Описание |
|------|----------|
| `--type {тип}` | Пропустить выбор типа (вебинар, конференция, митап...) |
| `--quick` | Быстрый режим — только обязательные шаги |
| `--lang en` | Генерация на английском языке |
| `--output {путь}` | Директория для сохранения файлов |

## Типы мероприятий

| Тип | Описание | Формат |
|-----|----------|--------|
| вебинар | Онлайн-трансляция | online |
| конференция | Масштабное мероприятие | hybrid |
| митап | Неформальная встреча | offline |
| воркшоп | Практическое занятие | offline |
| хакатон | Соревнование разработчиков | hybrid |
| демо-день | Презентация проектов | hybrid |
| партнёрское | Совместное мероприятие | varies |

## Навигация

| Команда | Действие |
|---------|----------|
| `назад` | Вернуться на шаг |
| `пропустить` | Пропустить опциональный шаг |
| `статус` | Показать текущие данные |
| `превью` | Посмотреть результат |
| `начать заново` | Перезапустить wizard |

## Результат

### Файлы

После завершения wizard сохраняет:
- `events/{slug}.md` — Markdown-страница
- `events/{slug}.json` — JSON-структура

### Markdown-страница

```markdown
# VK Process Mining 2.0: новые возможности

22 октября 2024 года

Начало события в 11:00

## О мероприятии

У VK Process Mining глобальное обновление...

**В программе:**
- Что нового в версии 2.0
- Как работает мультипроцессная аналитика
- Q&A

**Спикеры:**
- Александр Вилкул, старший менеджер продукта VK Tech
```

### JSON-структура

```json
{
  "event": {
    "type": "вебинар",
    "title": "VK Process Mining 2.0",
    "slug": "vk-process-mining-20",
    "date": "2024-10-22",
    "time": "11:00"
  },
  "speakers": [...],
  "program": [...]
}
```
