# «Русский Свет» перевёл всех сотрудников на коммуникационные сервисы от VK WorkSpace

Официальный дистрибьютор производителей электротехники «Русский Свет» перевёл 5 тысяч сотрудников на коммуникационную платформу от VK Tech. При поддержке ГК MONT и ИТ-интегратора 2Brains компания получила доступ к облачным сервисам VK WorkSpace: корпоративной почте, мессенджеру, а также файловому хранилищу с редактором документов.

Команде «Русского Света» требовалось облачное решение, которое позволило бы снизить затраты на оборудование и обеспечило бы надёжный уровень сервиса. Также было важно наличие у вендора распределённых дата-центров на территории России.

Дистрибьюторы MONT и ИТ-интегратор 2Brains помогли «Русскому Свету» остановить свой выбор на комплексном SaaS-решении от VK WorkSpace. За две недели партнёры вместе с техническими специалистами VK Tech перевели весь штат компании на новую платформу, не останавливая при этом бизнес-процессы. В рамках перехода сотрудникам была подключена корпоративная почта с календарём и адресной книгой, облачное хранилище со встроенным онлайн-редактором документов, корпоративный мессенджер.

*«Перед нами стояла важная задача по выбору российского решения, которое будет обеспечивать непрерывность работы почты и безопасность корпоративных коммуникаций. Более 200 тыс. писем отправляется нашими сотрудниками ежедневно. Поэтому для эффективной работы нам была необходима бизнес-почта с неограниченным объёмом ящика, простым интерфейсом, расширенными настройками безопасности и гибкими настройками администрирования»*, — отметил **начальник управления по инфраструктуре информационных технологий «Русского Света» Алексей Савченко**.

*«Мы оказываем профессиональные услуги по выбору облачных сервисов и поддерживаем клиента на всех этапах миграции. Для «Русского Света» нам удалось вместе с экспертами VK Tech в кратчайшие сроки настроить биллинг системы. Если заказчик в будущем по мере масштабирования бизнеса решит увеличить количество пользователей, то у него будет возможность сделать это достаточно оперативно»*, — подчеркнул **генеральный директор 2Brains Александр Чеглаков**.

*«Более 25 лет мы развиваем коммуникационные сервисы и накопили опыт в создании надёжных решений для клиентов в разных отраслях. Корпоративная почта остаётся важным каналом коммуникаций для бизнеса. Мы делаем всё, чтобы предоставить безопасное решение, которое отвечает требованиям заказчика и максимально бесшовно встраивается в его текущую ИТ-инфраструктуру»*, — прокомментировал **коммерческий директор VK WorkSpace Антон Тен**.

## Анализ структуры

| Элемент | Содержание |
|---------|------------|
| **Заголовок** | Клиент + действие + решение |
| **Лид** | Кто (Русский Свет) + Что (перевёл 5 тыс. сотрудников) + Партнёры (MONT, 2Brains) |
| **Задача** | Облачное решение, снижение затрат, дата-центры в России |
| **Процесс** | 2 недели, без остановки бизнес-процессов |
| **Цитата клиента** | Задача + масштаб (200 тыс. писем/день) + требования |
| **Цитата интегратора** | Процесс миграции + масштабируемость |
| **Цитата VK WorkSpace** | Опыт + надёжность + бесшовность |

## Ключевые особенности

1. **Три цитаты** — клиент + интегратор + VK WorkSpace
2. Конкретные метрики: 5 тыс. сотрудников, 200 тыс. писем/день, 2 недели
3. Нет бойлерплейта в конце (особенность кейсовых релизов)
4. Спикер VK WorkSpace — Антон Тен (коммерческий директор), не Щеглов
5. Нарратив: задача → выбор → внедрение → результат
