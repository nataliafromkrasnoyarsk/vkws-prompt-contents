# SEO-визард для VK WorkSpace

Запусти агента `vk-workspace-seo-wizard` для интерактивного создания SEO-статьи в блог `workspace.vk.ru/blog`.

## Использование

```
/seo-article-vk-workspace
```

Или с темой сразу:

```
/seo-article-vk-workspace Российские аналоги Microsoft Teams
```

Если в аргументах передана тема — используй её как стартовое значение для ШАГа 1 визарда и сразу переходи к ШАГу 2 (выбор продукта).

## Что делает визард

12-шаговый интерактивный поток:

1. Тип и тема статьи
2. Продукт VK WorkSpace в фокусе
3. Роль эксперта (ИТ-директор / продуктивность / ИБ / ЦТ)
4. Целевая аудитория
5. Дополнительная информация и метатеги
6. Ключевые слова (primary, secondary, long-tail, LSI)
7. Режим генерации (Balanced / Deep Expert / Light / Strict SEO / Brand)
8. Тон (Expert / Neutral / Technological)
9. Структура статьи
10. Подтверждение параметров
11. Генерация:
    - 11a — варианты H1/Title/Meta description (3–5 штук с обоснованием)
    - 11b — внутренние ссылки из реестра `internal-links.md`
    - 11c — генерация полной статьи
    - 11d — self-audit с автофиксом перед показом
12. Доработка и публикация (commit + push на текущую ветку)

## База знаний

Агент использует skill `vk-workspace-knowledge`:
- Каталог 13 продуктов VK WorkSpace
- 5 сегментов аудиторий (от микро до энтерпрайза)
- Конкуренты и позиционирование
- Tone of Voice блога
- SEO-стратегия и кластеры ключевых слов
- Антипаттерны (выдуманная статистика, штампы, шаблонные таблицы)
- **Реестр существующих статей блога** для internal linking (`internal-links.md`)

## Аргументы

| Аргумент | Описание |
|---|---|
| `$ARGUMENTS` | Опциональная тема статьи. Если задана — используется как стартовое значение для ШАГа 1 |

## Запуск

Передай управление агенту `vk-workspace-seo-wizard`. Если `$ARGUMENTS` непустой — начни визард с предзаполненной темой.
