# Zen Article Agent — VK WorkSpace

Интерактивный визард для создания статей о VK WorkSpace для канала в Яндекс Дзен.

## Запуск

```
/write-zen-article
```

Или с темой:

```
/write-zen-article Как выбрать корпоративный мессенджер
```

## Отличие от SEO-визарда

| Параметр | SEO-визард (блог) | Дзен-визард |
|----------|-------------------|-------------|
| Цель | Позиция в поиске | Дочитывания + охваты |
| Оптимизация | Ключевые слова, meta | CTR заголовка, удержание |
| Тон | Профессиональный | Разговорный, живой |
| CTA | «Попробуйте бесплатно» | «Подпишитесь / комментируйте» |
| Длина | 1500-3000 слов | 2500-3500 символов б/п |

## Типы статей

- **Образовательная** — «Что такое X и зачем это бизнесу»
- **Практический гайд** — «Как сделать X: пошаговая инструкция»
- **Сравнение / обзор** — «X vs Y: что выбрать»
- **Кейс / история** — «Как компания X решила проблему Y»
- **Новость** — «VK WorkSpace запустил X» / инфоповод + контекст + польза
- **Подборка / рейтинг** — «N лучших инструментов для...»
- **Тренд-статья** — «N трендов в ... на 2026 год»
- **Мнение / колонка** — Провокационный тезис + аргументация

## Режимы генерации

- **Engagement** — Максимум дочитываний (рекомендуется)
- **Expert** — Экспертная статья с аналитикой
- **Light** — Лёгкая и доступная
- **Story** — Нарративный формат через историю

## База знаний

### zen-knowledge
Содержит:
- Алгоритм Дзен и метрики ранжирования
- Формулы заголовков (10 формул)
- Приёмы оптимизации дочитываний
- Типы статей и структурные шаблоны
- Правила интеграции VK WorkSpace
- Требования модерации Дзен
- Правила работы с поисковыми запросами из ТЗ
- Оптимизация для AI-поисковиков (cite-safety, ETB-блоки)
- Чеклист качества статьи

## Общие требования

- **Объём:** 2500-3500 символов без пробелов (допуск ±10-20%)
- **Заголовок из ТЗ:** если в ТЗ указана тема, она используется как заголовок — менять нежелательно
- **Структура:** не более 5 заголовков H2 на статью
- **Поисковые запросы:** если указаны — каждый минимум 1 раз, равномерно по тексту, по возможности в H2
- **Cite-Safety (AI-поисковики):** ключевые блоки пишутся как ETB (Extractable Trust Blocks) — самодостаточные при цитировании AI-поисковиками. Нет слепых зон контекста, нет категоричности без границ

## Сохранение статей

Статьи сохраняются в:
```
content-hub/zen-articles/drafts/{YYYY-MM-DD}-{slug}.md
```

После одобрения перемещаются в:
```
content-hub/zen-articles/published/
```
