# Реестр статей блога VK WorkSpace

## Как пользоваться

При генерации статьи (ШАГ 11) подбери **2–5 внутренних ссылок** из этого реестра:

1. **Приоритет** — статьи с совпадающим модулем (продуктом)
2. **Второй приоритет** — статьи смежной тематики (импортозамещение, безопасность, коммуникации)
3. **Не ссылайся** на акционные/партнёрские материалы в редакционном контексте
4. **Anchor text** — формулируй описательно («подробнее о настройке корпоративной почты»), не «тут» или «по ссылке»
5. **Не ссылайся на саму себя** — исключай текущую статью, если slug совпадает

Типы статей: `обзор` — обзор-сравнение, `гайд` — инструкция, `обр` — образовательная, `продукт` — продуктовый обзор, `кейс` — история успеха/кейс
