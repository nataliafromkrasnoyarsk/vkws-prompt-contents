# Коллекция примеров для создания контента

Эта папка содержит примеры успешных публикаций в российских СМИ за 2024-2025 годы. Используй эти материалы как референсы при создании новых статей и колонок.

## Структура коллекции

```
examples/
├── README.md                    # Этот файл
├── best-articles-2024.md        # Общий обзор лучших практик
├── habr-examples.md             # Примеры для Хабра
├── vc-ru-examples.md            # Примеры для VC.ru
├── business-media-examples.md   # Forbes, РБК, Ведомости, Коммерсант
└── tech-portals-examples.md     # 3DNews, iXBT, Hi-Tech Mail.ru, Ferra
```

## Как использовать

### При создании статьи для Хабра
1. Изучи `habr-examples.md` — форматы, структуры, антипаттерны
2. Посмотри `best-articles-2024.md` — универсальные паттерны заголовков
3. Выбери подходящий формат (расследование, туториал, мнение, личный опыт)

### При создании колонки для Forbes/РБК
1. Изучи `business-media-examples.md` — требования и успешные примеры
2. Обрати внимание на структуру экспертной колонки
3. Подготовь данные и статистику из авторитетных источников

### При создании материала для VC.ru
1. Изучи `vc-ru-examples.md` — форматы и метрики
2. Выбери раздел (AI, Бизнес, Мнения, Стартапы)
3. Учти оптимальное время публикации

### При создании обзора для tech-порталов
1. Изучи `tech-portals-examples.md` — структуры обзоров
2. Подготовь визуальный контент
3. Следуй требованиям конкретного портала

## Ключевые инсайты

### Универсальные паттерны успеха

**Заголовки:**
- Число + конкретика: «5 IT-профессий, которые выстрелят в 2024»
- Интрига + бренд: «Ваш Xiaomi — это не Xiaomi»
- Личная история: «Как я собеседовался в Ozon, Т-Банк...»

**Структура:**
- Сильный hook в начале
- Чёткие разделы с подзаголовками
- Данные и примеры
- Практические выводы

**Элементы доверия:**
- Ссылки на первоисточники
- Конкретные цифры
- Экспертный статус автора
