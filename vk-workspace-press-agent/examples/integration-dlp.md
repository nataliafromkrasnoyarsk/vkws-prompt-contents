# Пример: Интеграция с DLP-системой

Эталонный пресс-релиз об интеграции VK WorkSpace с InfoWatch Traffic Monitor.

# VK WorkSpace и DLP-система InfoWatch Traffic Monitor защитят от утечек данных

Коммуникационная платформа VK WorkSpace в on-premise версии теперь совместима с системой предотвращения утечек данных (DLP) InfoWatch Traffic Monitor. Это позволит контролировать передачу чувствительных данных организации и предотвращать их потенциальное несанкционированное распространение через корпоративную почту и мессенджер.

Интеграция VK WorkSpace с InfoWatch Traffic Monitor обеспечивает комплексную защиту всех видов рабочих коммуникаций в основных каналах передачи данных — корпоративном мессенджере и электронной почте — через приложение VK Teams и веб-интерфейс.

В мессенджере VK WorkSpace DLP-система InfoWatch анализирует коммуникации: текстовые сообщения, вложённые документы и файлы в корпоративной переписке. Система выявляет маркеры конфиденциальной информации в диалоге и фиксирует нарушения политики работы с данными организации. При выявлении нарушения политики информационной безопасности система автоматически блокирует отправку сообщения, ограничивает доступ к файлам и уведомляет о нарушении специалистов по информационной безопасности.

В рамках платформы VK WorkSpace электронная почта также интегрируется с DLP-системой InfoWatch Traffic Monitor. Предусмотрено несколько сценариев взаимодействия с DLP-системой, включая подачу почтового трафика через SMTP-relay и в скрытой копии (BCC). Для настройки интеграции достаточно указать адрес сервера DLP или почтовый ящик в административном интерфейсе VK WorkSpace.

*«Мы непрерывно развиваем наши решения, учитывая потребности заказчиков и специфику отечественного ИТ-рынка. Интеграция с системой InfoWatch Traffic Monitor — это новый шаг, помогающий нашим клиентам усилить защиту конфиденциальных данных и предотвратить риски, связанные с утечками»*, — отметил **директор по информационной безопасности VK Tech Анатолий Тутынин**.

*«Постоянное расширение поддержки российских платформ — одно из стратегических направлений для укрепления рыночного лидерства DLP-системы InfoWatch Traffic Monitor. VK WorkSpace — перспективная платформа, которая успешно замещает решения западных вендоров, сочетает мессенджер и электронную почту — основные средства корпоративных коммуникаций»*, — прокомментировал **директор департамента развития продуктов ГК InfoWatch Рустам Фаррахов**.

## Анализ структуры

| Элемент | Содержание |
|---------|------------|
| **Заголовок** | Два продукта + действие (защитят) + ценность (от утечек данных) |
| **Лид** | Совместимость + что контролирует + через какие каналы |
| **Мессенджер** | Анализ коммуникаций, маркеры, блокировка, уведомление ИБ |
| **Почта** | SMTP-relay, BCC, настройка через админпанель |
| **Цитата VK Tech** | Развитие, потребности заказчиков, усиление защиты |
| **Цитата партнёра** | Стратегия поддержки российских платформ, импортозамещение |

## Ключевые особенности

1. **Две цитаты** — VK Tech (Тутынин, ИБ) + партнёр (InfoWatch)
2. **Технический фокус** — SMTP-relay, BCC, pushAPI SDK
3. **Спикер VK** — Тутынин (директор по ИБ), не Щеглов
4. **Расшифровка DLP** — «система предотвращения утечек данных» в первом упоминании
5. **Конкретные сценарии** — мессенджер + почта через два канала
6. **Нет бойлерплейтов** — укороченный формат
