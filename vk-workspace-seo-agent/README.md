# VK WorkSpace SEO Agent

Интерактивный визард для создания SEO-оптимизированных статей для блога VK WorkSpace (workspace.vk.ru/blog).

## Запуск

```
/seo-article-vk-workspace
```

Или с темой:

```
/seo-article-vk-workspace Российские аналоги Microsoft Teams
```

## База знаний

### Skill `vk-workspace-knowledge` (`SKILL.md`)
- Каталог всех продуктов VK WorkSpace (13 сервисов)
- Целевые аудитории (5 сегментов: от микро до энтерпрайз)
- Конкуренты и позиционирование
- Безопасность и compliance
- Терминология и правила написания
- SEO-стратегия и кластеры ключевых слов
- Tone of Voice и стиль блога
- Типы статей и шаблоны структур
- Режимы генерации
- Антипаттерны (выдуманная статистика, штампы, шаблонные таблицы)

### Реестр статей блога (`internal-links.md`)
- 81 опубликованная статья с workspace.vk.ru/blog
- Сгруппировано по 9 разделам (Почта, Мессенджер, ВКС, Диск и Документы, Доска, Проекты, Задачи и Заметки, Календарь, Суперапп, Платформа)
- Каждая статья помечена типом (`обзор`, `гайд`, `обр`, `продукт`, `кейс`)
- Дополнительные тематические кластеры (импортозамещение, удалёнка, безопасность, управление проектами)
- Используется визардом в ШАГе 11b для подбора 2–5 внутренних ссылок

## Self-audit (ШАГ 11d)

Перед показом статьи пользователю визард автоматически проверяет и **молча исправляет**:

1. Плотность primary KW (1–2%)
2. Длину Meta Title (≤60 симв.)
3. Длину Meta Description (150–160 симв.)
4. Наличие KW в H1
5. Наличие KW в первом абзаце (≤100 слов)
6. Наличие KW хотя бы в одном H2
7. Длину абзацев (≤4 предложений)
8. Штампы («уникальный», «инновационный», «В современном мире» и др.)
9. Развёрнутый CTA (3–5 предложений)
10. FAQ как реальные поисковые запросы
11. Наличие 2–5 internal links из реестра

После проверки выводится краткий отчёт аудита, затем — текст статьи.

## Выходные файлы

Статьи сохраняются в:
```
content-hub/vk-workspace-articles/drafts/{YYYY-MM-DD}-{slug}.md
content-hub/vk-workspace-articles/published/{YYYY-MM-DD}-{slug}.md
```

## Поддержка реестра в актуальном состоянии

Реестр `internal-links.md` обновляется вручную при выходе новых статей. Источник данных — таблица «VK WorkSpace blog articles» (Google Sheets). Рекомендуется синхронизировать раз в месяц.
