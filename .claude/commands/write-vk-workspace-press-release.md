# Press Release Wizard — VK WorkSpace

Запусти интерактивный визард для написания пресс-релиза VK WorkSpace.

## Инструкции

1. Загрузи скиллы:
   - `vk-workspace-press-knowledge` — бойлерплейты, спикеры, цитаты, терминология, правила написания, чеклист качества
   - `vk-workspace-knowledge` — база знаний о платформе VK WorkSpace: 13 продуктов, позиционирование, аудитории, конкуренты, терминология

2. Действуй как агент `vk-workspace-press-wizard` — проведи пользователя через 10 шагов:
   - Шаг 1: Приём материалов
   - Шаг 2: Анализ + Тип релиза (запуск, обновление, кейс, интеграция, ИБ, рейтинг, исследование, тарифы)
   - Шаг 3: Продукт VK WorkSpace в фокусе
   - Шаг 4: Исследование (опционально)
   - Шаг 5: Уточняющие вопросы (спикер, цифры, партнёр, безопасность)
   - Шаг 6: Варианты заголовков (3 варианта → выбор)
   - Шаг 7: Лид-абзац + Описание продукта
   - Шаг 8: Генерация основного текста (цитата, детали, бойлерплейт)
   - Шаг 9: Сборка, валидация и сохранение (⚠️ показать текст, сохранить файл, спросить о доработках)
   - Шаг 10: Ревью и доработка (⚠️ только после подтверждения → коммит + пуш)

3. Сохраняй в `content-hub/press-releases/drafts/`

4. **ВАЖНО:** Не путай VK WorkSpace с VK Cloud — это разные продукты!

## Аргумент $ARGUMENTS

Если передан аргумент, используй его как описание темы релиза и пропусти Шаг 1.
