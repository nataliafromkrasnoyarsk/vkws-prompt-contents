# VK WorkSpace Press Release Knowledge Skill

Правила, шаблоны и примеры для написания пресс-релизов VK WorkSpace, основанные на анализе 13 реальных релизов tech.vk.com.

## АНАТОМИЯ ПРЕСС-РЕЛИЗА VK WORKSPACE

### Структура

```
┌─────────────────────────────────────────────────────────┐
│  ЗАГОЛОВОК (H1)                                         │
│  Компания + действие + продукт + ценность               │
├─────────────────────────────────────────────────────────┤
│  ЛИД-АБЗАЦ                                              │
│  Что + Зачем + Контекст (2-3 предложения)               │
├─────────────────────────────────────────────────────────┤
│  ОПИСАНИЕ ПРОДУКТА / ОБНОВЛЕНИЯ                         │
│  Функционал, возможности, сценарии                      │
│  (сплошным текстом или с подзаголовками по блокам)       │
├─────────────────────────────────────────────────────────┤
│  ЦИТАТА СПИКЕРА                                         │
│  Рынок → Проблема → Решение → Ценность                  │
├─────────────────────────────────────────────────────────┤
│  ДОПОЛНИТЕЛЬНЫЕ ДЕТАЛИ (опционально)                    │
│  Доступность, тарифы, сертификация, партнёры            │
├─────────────────────────────────────────────────────────┤
│  ПАРТНЁРСКИЕ ЦИТАТЫ (если есть)                         │
│  Цитата от клиента / интегратора / партнёра             │
├─────────────────────────────────────────────────────────┤
│  БОЙЛЕРПЛЕЙТЫ                                           │
│  О VK Tech + О VK WorkSpace                             │
└─────────────────────────────────────────────────────────┘
```

### Особенности стиля VK WorkSpace

В отличие от VK Cloud, пресс-релизы VK WorkSpace:
- Могут использовать **подзаголовки внутри текста** для структурирования крупных обновлений
- Фокусируются на **корпоративных коммуникациях**, а не на облачной инфраструктуре
- Оперируют терминологией **рабочего пространства** (мессенджер, ВКС, почта, диск, проекты)
- Часто упоминают **импортозамещение** и **российские ОС** (Ред ОС, Astra Linux)
- Подчёркивают **безопасность данных** и **контроль в контуре заказчика**

## ТИПЫ ПРЕСС-РЕЛИЗОВ VK WORKSPACE

| Тип | Характеристики | Типичная структура |
|-----|----------------|-------------------|
| **Запуск продукта** | Новый сервис в составе платформы | Лид → Описание → Возможности → Цитата → Бойлерплейт |
| **Обновление платформы** | Новая версия (25.3, 25.4) | Лид → Блоки по сервисам с подзаголовками → Цитата → Доступность → Бойлерплейт |
| **Кейс клиента** | Внедрение у заказчика | Лид → Задача клиента → Решение → Результат → Цитаты (клиент + VK) → Бойлерплейт |
| **Интеграция / фича** | Новая функциональность | Лид → Как работает → Цитата → Бойлерплейт |
| **Рейтинг / признание** | Место в рейтинге CNews, TAdviser | Лид → Методология рейтинга → Результаты → Возможности продукта → Бойлерплейт |
| **Исследование / аналитика** | Данные об использовании | Лид → Методология → Ключевые находки → Цитата → Бойлерплейт |
| **Безопасность** | Блокировка угроз, антиспам | Лид → Масштаб угроз → Типы атак → Технологии защиты → Цитата → Бойлерплейт |
| **Тарифы / коммерция** | Новая линейка тарифов | Лид → Обзор линейки → Описание каждого тарифа → Условия перехода |

| Контекст | Глаголы |

## БОЙЛЕРПЛЕЙТЫ КОМПАНИЙ

### VK Tech (полный)

```markdown
**VK Tech** — российский разработчик корпоративного ПО для решения ежедневных
задач бизнеса. Портфель VK Tech — это готовая экосистема программных продуктов,
которая включает облачную платформу, дата-сервисы, сервисы продуктивности
и бизнес-приложения. Продукты VK Tech включены в реестр российского ПО
и соответствуют требованиям ФСТЭК, поставляются в формате лицензий (On-Premises)
и по подписке (On-Cloud), что делает их доступными для компаний любого масштаба.
```

### VK WorkSpace (полный)

```markdown
**VK WorkSpace** — платформа для корпоративных коммуникаций и совместной работы
команд любого масштаба, где все сервисы интегрированы между собой и доступны
через сквозные сценарии в едином супераппе. Включает в себя корпоративную почту
с календарём и адресной книгой, мессенджер, видеоконференции, файловое хранилище
со встроенным редактором документов, опросы, заметки, задачи и доску.
Все сервисы находятся в едином пространстве, доступны в режиме одного окна
и управляются из общей административной панели. У продуктов несколько независимых
слоёв защиты. Платформу можно развернуть на серверах компании (On-Premises)
или использовать в облаке (SaaS). Все сервисы входят в реестр российского ПО
и отвечают требованиям ФЗ-152.
```

## ПРАВИЛА НАПИСАНИЯ

### 1. Структурирование больших обновлений

В отличие от VK Cloud, пресс-релизы VK WorkSpace об обновлениях платформы могут использовать **подзаголовки** для разделения блоков по сервисам:

```markdown
**Новый суперапп: модульная архитектура, работа без интернета и ИБ**

{текст о супераппе}

**Геораспределённая Почта для федеральных компаний**

{текст о почте}

**Мессенджер: общение между организациями**

{текст о мессенджере}
```

### 2. Расшифровка терминов для журналистов

Сложные IT-термины нужно объяснять простым языком:

```
Суперапп VK WorkSpace собирает в одном интерфейсе сервисы
платформы: Почту, Мессенджер, Календарь, Видеозвонки, Диск
с Документами, Оргструктуру и Опросы.
```

### 3. Без буллитов в тексте релиза (предпочтительно)

Текст пресс-релиза предпочтительно писать сплошным текстом. Допустимы подзаголовки и жирные выделения продуктов в релизах о линейке. Маркированные списки — только в описании тарифов и рейтинговых таблицах.

### 4. Кейсы клиентов: три цитаты

В релизах о внедрениях у клиентов используется формат трёх цитат:
1. **Цитата клиента** — о задаче и выборе решения
2. **Цитата интегратора** (если есть) — о процессе внедрения
3. **Цитата VK WorkSpace** — об экспертизе и результате
