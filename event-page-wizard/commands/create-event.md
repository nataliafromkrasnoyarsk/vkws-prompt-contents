# /event-page-wizard:create-event

Запускает пошаговый wizard для создания страницы мероприятия.

## Usage

```
/event-page-wizard:create-event
```

### С параметрами

```bash
# Быстрое создание вебинара
/event-page-wizard:create-event --type вебинар --quick

# Конференция с сохранением в папку
/event-page-wizard:create-event --type конференция --output ./events/

# На английском языке
/event-page-wizard:create-event --lang en
```

### Естественный язык

Также можно написать:
- "Создай страницу вебинара"
- "Страница конференции"
- "Создать ивент"
- "Хакатон страница"

## Wizard Steps

```
┌─────────────────────────────────────────────────────────┐
│  Step 1:  Тип мероприятия                    [ОБЯЗАТ.] │
│  Step 2:  Название                           [ОБЯЗАТ.] │
│  Step 3:  Дата и время                       [ОБЯЗАТ.] │
│  Step 4:  Формат и место                     [ОБЯЗАТ.] │
│  Step 5:  Описание                           [ОБЯЗАТ.] │
│  Step 6:  Программа                          [ОПЦИОН.] │
│  Step 7:  Спикеры                            [ОБЯЗАТ.] │
│  Step 8:  Целевая аудитория                  [ОПЦИОН.] │
│  Step 9:  Партнёры                           [ОПЦИОН.] │
│  Step 10: Дополнительно                      [ОПЦИОН.] │
├─────────────────────────────────────────────────────────┤
│  ФИНАЛ:   Превью и сохранение                          │
└─────────────────────────────────────────────────────────┘
```

> При использовании `--quick` пропускаются шаги 6, 8, 9, 10

## Event Types

| Тип | Описание | Формат по умолчанию |
|-----|----------|---------------------|
| вебинар | Онлайн-трансляция | online |
| конференция | Масштабное мероприятие | hybrid |
| митап | Неформальная встреча | offline |
| воркшоп | Практическое занятие | offline |
| хакатон | Соревнование разработчиков | hybrid |
| демо-день | Презентация проектов | hybrid |
| партнёрское | С партнёром | varies |

## Navigation Commands

| Команда | Действие |
|---------|----------|
| `назад` | Вернуться на шаг назад |
| `пропустить` | Пропустить опциональный шаг |
| `статус` | Показать текущие данные |
| `начать заново` | Перезапустить wizard |
| `помощь` | Показать справку |
| `превью` | Посмотреть результат |

## Output

После завершения wizard генерирует и сохраняет:

### 1. Markdown-страница (`events/{slug}.md`)
Готовая страница для сайта в формате Markdown.

### 2. JSON-структура (`events/{slug}.json`)
```json
{
  "event": {
    "type": "вебинар",
    "title": "Название",
    "slug": "nazvanie",
    "date": "2024-10-22",
    "time": "11:00"
  },
  "speakers": [...],
  "program": [...],
  "partners": [...]
}
```
