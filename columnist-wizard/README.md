## Использование

### Интерактивный визард
```
/write-column
```
Запустит пошаговый процесс с выбором через кнопки:
1. Выбор персоны (автора) `[1-7]`
2. Выбор издания → загрузка skill `[A-N]`
3. Указание источника материала `[1-5]`
4. Выбор типа колонки `[1-5]`
5. Создание и согласование плана
6. Написание и ревью колонки
7. Выбор места сохранения `[1-5]`

### Прямой запрос
```
Напиши колонку про [тема] для [издание] от имени [автор]
```

## Типы колонок

1. **Трендовый анализ** — что меняется на рынке
2. **Контринтуитивный взгляд** — вызов общепринятому мнению
3. **Из опыта** — уроки реальных проектов
4. **Стратегический комментарий** — реакция на события
5. **Методология / How-To** — практические фреймворки

## Сохранение колонок

После финализации колонки доступны варианты сохранения:

| Опция | Описание | Путь |
|-------|----------|------|
| `[1]` Markdown | Файл .md для редактирования | `drafts/[издание]-[дата]-[slug].md` |
| `[2]` DOCX | Документ Word для редакции | `drafts/[издание]-[дата]-[slug].docx` |
| `[3]` База знаний | Пример в skill издания | `.claude/skills/media/[категория]/[издание].md` |
| `[4]` Комбинация | Несколько вариантов | По выбору (например, "1,3") |
| `[5]` Не сохранять | Оставить в чате | — |

### Структура директории drafts
```
drafts/
├── forbes-2025-01-15-cloud-security.md
├── forbes-2025-01-15-cloud-security.docx
├── rbc-2025-01-20-digital-transformation.md
└── ...
```

### Добавление в базу знаний
При выборе опции `[3]` колонка добавляется как пример в skill издания. Это улучшает качество будущих статей для этого СМИ.

## Ключевые принципы

### Критические правила
- **Ноль маркетинга** — никакого продвижения продуктов VK Cloud
- **Источники обязательны** — все данные должны быть подтверждены
- **Российский контекст** — всегда связываем с местным рынком
- **Согласование плана** — не писать без одобрения структуры

### Что делает колонку успешной
1. Авторитет без высокомерия
2. Практическая ценность для читателя
3. Нативность для издания
4. Честность об ограничениях
5. Данные и примеры
