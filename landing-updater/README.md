# Landing Page Updater

Интерактивный агент для обновления текстов лендингов VK Cloud с сохранением структуры и SEO-оптимизацией.

## Что делает

1. **Получает текст** — по ссылке, из файла или напрямую
2. **Разбирает на блоки** — автоматическая сегментация с определением типов
3. **Применяет SEO** — ваши ключи или предложения агента
4. **Учитывает референсы** — страница-образец или текстовые указания
5. **Предлагает улучшения** — анализ каждого блока с рекомендациями
6. **Генерирует текст** — обновлённый контент с сохранением структуры

## Ключевой принцип

```
СОХРАНЯЕМ:                 ОБНОВЛЯЕМ:
- Количество блоков        - Формулировки
- Количество предложений   - Актуальность данных
- Иерархию H1->H2->текст   - SEO-оптимизацию
- Общий смысл              - Устаревшие термины
```

## Установка

### Вариант 1: Копирование в проект

Скопируйте содержимое папки `.claude/` в ваш проект:

```bash
cp -r landing-updater/.claude/ /path/to/your/project/.claude/
```

### Вариант 2: Символическая ссылка

```bash
ln -s /path/to/landing-updater/.claude/agents/landing-updater-wizard.md \
      /path/to/your/project/.claude/agents/

ln -s /path/to/landing-updater/.claude/commands/update-landing.md \
      /path/to/your/project/.claude/commands/

ln -s /path/to/landing-updater/.claude/skills/landing-knowledge.md \
      /path/to/your/project/.claude/skills/
```

## Быстрый старт

### Запуск команды

```
/update-landing https://vkcloud.kz/containers/
```

Просто напишите в чате:

## SEO-параметры

| Параметр | Описание |
|----------|----------|
| **Основные ключи** | Главные ключевые слова |
| **LSI-ключи** | Дополнительные/связанные слова |
| **Title** | Meta-заголовок (до 60 символов) |
| **Description** | Meta-описание (до 160 символов) |

Можете указать свои или попросить агента предложить на основе анализа страницы.

## Референсы и указания

### Варианты ввода

1. **URL референса** — страница с желаемым стилем/тоном
2. **Текстовые указания** — что конкретно изменить
3. **Оба варианта** — и референс, и инструкции
4. **Пропустить** — обновление на основе SEO и best practices

### Примеры указаний

- "Сделать текст более динамичным"
- "Добавить упоминание ИИ и ML"
- "Обновить цифры по клиентам"
- "Добавить акцент на безопасность"
- "Убрать упоминание конкретного региона"

## Команды

### Навигация

| Команда | Действие |
|---------|----------|
| `назад` | Вернуться к предыдущему этапу |
| `статус` | Показать текущее состояние |
| `блок N` | Перейти к редактированию блока N |

### Доработка результата

| Команда | Действие |
|---------|----------|
| `короче блок N` | Сократить конкретный блок |
| `подробнее блок N` | Расширить конкретный блок |
| `больше SEO` | Добавить больше ключевых слов |
| `меньше SEO` | Убрать избыток ключей |
| `вернуть блок N` | Откатить блок к исходному |
| `экспорт` | Показать чистый текст для копирования |

## База знаний VK Cloud

Skill `landing-knowledge` содержит:

- **Типовую структуру** лендингов VK Cloud (Hero, Features, Solutions...)
- **Шаблоны блоков** с примерами из реальных страниц
- **Правила SEO** — размещение ключей, meta-теги
- **Словарь стиля** — DO/DON'T с примерами трансформации
- **Чеклисты** для проверки качества

## Используемые инструменты

Агент работает с:

| Tool | Назначение |
|------|------------|
| `WebFetch` | Загрузка страниц по URL |
| `Read` | Чтение локальных файлов |
| `Grep` | Поиск контента |
| `Write` | Сохранение результата |

## Ограничения

- **JavaScript-контент**: Страницы с динамической загрузкой могут не парситься полностью
- **Авторизация**: Закрытые страницы недоступны — используйте копирование текста
- **Размер**: Очень большие страницы (>50KB текста) могут обрабатываться частями

## На основе анализа

Агент создан на основе анализа реальных страниц:

- Главная страница vkcloud.kz
- Страницы продуктов (Containers, Storage, Databases)
- Промо-страницы и акции
- Документация VK Cloud
