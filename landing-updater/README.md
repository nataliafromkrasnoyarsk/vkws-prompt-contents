# Landing Page Updater

Интерактивный агент для обновления текстов лендингов VK Cloud с сохранением структуры и SEO-оптимизацией.

## 🎯 Что делает

1. **Получает текст** — по ссылке или напрямую
2. **Разбирает на блоки** — автоматическая сегментация
3. **Применяет SEO** — ваши ключи или предложения агента
4. **Учитывает референсы** — страница-образец или указания
5. **Предлагает улучшения** — анализ каждого блока
6. **Генерирует текст** — с сохранением структуры

## 🔑 Ключевой принцип

```
СОХРАНЯЕМ:              ОБНОВЛЯЕМ:
✓ Количество блоков     ✓ Формулировки
✓ Количество предложений ✓ Актуальность данных
✓ Иерархию H1→H2→текст  ✓ SEO-оптимизацию
✓ Общий смысл           ✓ Устаревшие термины
```

## 🚀 Быстрый старт

### Установка

```bash
/plugin marketplace add levashove/agents
/plugin install landing-updater
```

### Запуск

```bash
/landing-updater:update-landing https://example.com/page
```

Или просто:
```
Обнови лендинг https://vkcloud.kz/containers/
```

## 🔍 SEO-параметры

| Параметр | Описание |
|----------|----------|
| **Основные ключи** | Главные ключевые слова |
| **LSI-ключи** | Дополнительные слова |
| **Title** | Meta-заголовок (до 60 символов) |
| **Description** | Meta-описание (до 160 символов) |

Можете указать свои или попросить агента предложить.

## 📖 Референсы и указания

### Варианты ввода

1. **URL референса** — страница с желаемым стилем/тоном
2. **Текстовые указания** — что изменить
3. **Оба варианта** — и референс, и инструкции
4. **Пропустить** — обновление на основе SEO

### Примеры указаний

- "Сделать текст более динамичным"
- "Добавить упоминание ИИ и ML"
- "Обновить цифры по клиентам"
- "Добавить акцент на безопасность"

## 🔧 Команды

### Навигация

| Команда | Действие |
|---------|----------|
| `назад` | Предыдущий этап |
| `статус` | Текущее состояние |
| `блок N` | Редактировать блок N |

### Доработка

| Команда | Действие |
|---------|----------|
| `короче блок N` | Сократить блок |
| `подробнее блок N` | Расширить блок |
| `больше SEO` | Добавить ключей |
| `меньше SEO` | Убрать избыток |
| `вернуть блок N` | Откатить к исходному |
