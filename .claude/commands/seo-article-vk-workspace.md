# SEO Article Wizard — VK WorkSpace

Запусти интерактивный визард для создания SEO-оптимизированной статьи для блога VK WorkSpace (workspace.vk.ru/blog).

## Инструкции

1. Загрузи скилл `vk-workspace-knowledge` — это твоя база знаний о платформе VK WorkSpace: продукты, позиционирование, целевые аудитории, конкуренты, SEO-стратегия, tone of voice.

2. Действуй как агент `vk-workspace-seo-wizard` — проведи пользователя через 13 шагов:
   - Шаг 1: Тема статьи (ОБЯЗАТЕЛЬНО)
   - Шаг 2: Продукт VK WorkSpace (ОБЯЗАТЕЛЬНО) — Почта, Календарь, Диск, Документы, ВКС, Мессенджер, Проекты, Доска, Суперапп, вся платформа
   - Шаг 3: Роль эксперта (ОБЯЗАТЕЛЬНО) — ИТ-директор, Специалист по продуктивности, Эксперт по ИБ, Специалист по цифровой трансформации, Универсальный
   - Шаг 4: Целевая аудитория (ОБЯЗАТЕЛЬНО) — ИТ-директора, Системные администраторы, Владельцы SMB, Разработчики, Широкая
   - Шаг 5: Ключевые слова (ОБЯЗАТЕЛЬНО)
   - Шаг 6: Режим генерации (ОБЯЗАТЕЛЬНО) — Balanced, Deep Expert, Light, Strict SEO, Brand
   - Шаг 7: Тон (ОБЯЗАТЕЛЬНО) — Expert, Neutral, Technological
   - Шаг 8: Структура (ОПЦИОНАЛЬНО)
   - Шаг 9: Ваша информация (ОПЦИОНАЛЬНО)
   - Шаг 10: Подтверждение параметров
   - Шаг 11: Генерация статьи → сохранение локально → показ в диалоге
   - Шаг 12: Цикл доработки — правки, обновление, повторный показ
   - Шаг 13: Сохранение → коммит и пуш только после подтверждения

3. На каждом обязательном шаге используй `AskUserQuestion` для выбора параметров.

4. При генерации статьи соблюдай:
   - SEO: H1 с ключом, meta title/description, H2/H3 с ключами, FAQ для featured snippets
   - Контент: 1500-3000 слов, структурированность, таблицы, списки
   - Tone of Voice: профессиональный, консультативный, без штампов
   - VK WorkSpace: органичное упоминание через сценарии использования
   - CTA: "Попробуйте бесплатно 30 дней" или "Запросите демо"

5. Сохраняй статьи в `content-hub/vk-workspace-articles/drafts/{YYYY-MM-DD}-{slug}.md`

## Аргумент $ARGUMENTS

Если передан аргумент, используй его как тему статьи и пропусти Шаг 1.
