# Компаниям не нужен «ещё один сервис для звонков»: кто и почему выиграл в рейтинге платформ видеоконференцсвязи

TAdviser опубликовал рейтинг крупнейших игроков российского рынка видеоконференцсвязи за 2024 год. Главный вывод: растут не те, кто делает «ещё один сервис для звонков». Растут те, кто предлагает единую рабочую среду, где видеозвонки — лишь один из инструментов, связанных с почтой, мессенджером, документами и задачами.

## Рынок разворачивается: от «просто звонков» к платформам

Аналитики TAdviser зафиксировали сдвиг: рынок уходит **от отдельных сервисов для видеозвонков к комплексным платформам** — то есть к решениям, которые объединяют в одной среде видеозвонки, мессенджер, почту, облачное хранилище, редактор документов и управление задачами.

Почему это происходит? Причин несколько — и все они понятны любому, кто работает в компании от 100 человек.

**Наследие импортозамещения.** Когда зарубежные сервисы ушли, компании начали закрывать дыры по одной: для звонков — одно решение, для почты — другое, для чатов — третье, для документов — четвёртое. Быстро, но хаотично.

**«Лоскутная» инфраструктура.** В итоге у типичной компании — **5-7 разрозненных сервисов** от разных вендоров. У каждого свой интерфейс, своя авторизация, своя логика. Сотрудники тратят время на переключение между окнами, а ИТ-отдел — на поддержку интеграций, которые ломаются при каждом обновлении.

**Усталость от «зоопарка».** По данным нашего исследования (750 компаний), **34% бизнесов уже выбирают комплексные решения от одного поставщика**. Причина простая — меньше интеграционных проблем, единая авторизация, предсказуемые затраты. Когда все инструменты работают вместе «из коробки», исчезает целый класс задач, которые раньше решал ИТ-отдел вручную.

**Видеозвонки — внутри рабочих процессов.** TAdviser фиксирует второй тренд — **интеграцию видеоконференцсвязи с бизнес-системами заказчика**. Компаниям нужны не звонки ради звонков, а звонки, встроенные в процессы: обсудили задачу в чате, перешли в видеозвонок, зафиксировали решение в проекте — без переключения между приложениями, без потери контекста.

## Что это значит для рынка

Аналитики прогнозируют: **к 2030 году на рынке видеоконференцсвязи в России останутся 1-2 крупных игрока**. Консолидация уже идёт.

У платформенных решений здесь преимущество. Отдельные сервисы для видеозвонков конкурируют только по качеству связи — а эту задачу сегодня закрывают практически все. Платформы конкурируют по ширине сценариев: чем больше рабочих задач закрывает одна среда, тем сложнее с неё уйти и тем лояльнее клиент.

Рейтинг TAdviser это подтверждает: растут те, кто предлагает не «ещё один сервис для звонков», а рабочую среду целиком.
