# SEO Content Creator Plugin v4.0

Интерактивный агент для создания SEO-статей с пошаговым заполнением параметров, выбором роли эксперта и гибким сохранением результата.

## Что делает плагин

Проводит пользователя через **10-шаговый wizard**, собирая параметры для генерации качественной SEO-статьи. Поддерживает выбор экспертной роли автора, ввод собственной информации по теме и сохранение в различных форматах.

## Быстрый старт

### Автоматический запуск

Просто напишите одну из фраз:
- "Создай статью"
- "Напиши статью"
- "Новая статья"
- "SEO статья"

### Через команду

```bash
/seo-content-creator:create-article
```

или глобально:

```bash
/seo-create-article
```

## Эксперты

Доступные роли экспертов:

| Эксперт | Описание |
|---------|----------|
| `vk-cloud-security` | Эксперт по ИБ VK Cloud (сертификации, compliance, защита инфраструктуры) |
| `general` | Универсальный автор (по умолчанию) |

Эксперты определяют терминологию, глубину и стиль статьи.

## Сохранение результата

После генерации статьи доступны варианты сохранения:

| Опция | Описание |
|-------|----------|
| `markdown` | Сохранить в .md файл |
| `html` | Сохранить в .html файл |
| `content-hub` | Сохранить в content-hub/drafts/ |
| `clipboard` | Скопировать в буфер обмена |
| `все` | Сохранить во все форматы |
| `не сохранять` | Пропустить (по умолчанию) |

## Навигация

В любой момент можно использовать команды:

| Команда | Действие |
|---------|----------|
| `назад` | Вернуться на шаг назад |
| `пропустить` | Пропустить опциональный шаг |
| `статус` | Показать текущие параметры |
| `начать заново` | Перезапустить wizard |
| `помощь` | Показать справку |
| `отмена` | Отменить и выйти |

## Параметры

### Режимы (Step 5)

| Режим | Описание |
|-------|----------|
| `balanced` | Стандартный (по умолчанию) |
| `deep_expert` | Технический экспертный |
| `light` | Упрощённый для всех |
| `strict_seo` | Максимум SEO |
| `brand_heavy` | Акцент на бренд |

### Тон (Step 6)

| Тон | Описание |
|-----|----------|
| `expert` | Профессиональный |
| `neutral` | Нейтральный |
| `technological` | Технический |

### Аудитория (Step 6)

| Сегмент | Описание |
|---------|----------|
| `SMB` | Малый/средний бизнес |
| `enterprise` | Крупный бизнес |
| `developers` | Разработчики |
| `marketers` | Маркетологи |
| `general` | Широкая аудитория |

### Ограничения (Step 8)

- `no_promo` — без рекламы
- `no_comparison` — без сравнений
- `no_hype` — без преувеличений
- `no_pricing` — без цен

## Результат

После завершения wizard вы получите:

```
Meta-данные
├── Title: SEO заголовок (до 60 символов)
├── Description: Мета-описание (150-160 символов)
└── URL: seo-friendly-url

Варианты H1
├── 1. Вариант с ключом
├── 2. Вариант с вопросом
└── 3. Вариант с годом

Статья
└── Полный текст в Markdown

Статистика
├── Слов: 2500
├── Ключей: 15
└── LSI: 20

Заметки для редактора

Сохранение (опционально)
├── ./articles/{slug}.md
├── ./articles/{slug}.html
└── content-hub/drafts/{slug}.md
```

## Триггеры автозапуска

Агент автоматически активируется на фразы:
- "seo статья", "создать seo статью"
- "написать статью", "создать статью"
- "напиши статью", "создай статью"
- "статья для блога", "новая статья"
- "генерация статьи", "сгенерировать статью"

## Связанные команды

- `/seo-content-creator:create-article` — запуск SEO wizard
- `/article-workflow` — полный workflow создания статьи
- `/draft-article` — создание черновика статьи
- `/trend-research` — исследование трендов
