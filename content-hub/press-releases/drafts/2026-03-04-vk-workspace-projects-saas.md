# VK Tech представил облачную версию решения для управления проектами в VK WorkSpace

Решение Проекты VK WorkSpace теперь доступно в облачной версии. Ранее сервис для управления задачами, проектами и стратегическими целями можно было развернуть только в контуре компании — теперь его могут подключить и пользователи облачной подписки VK WorkSpace. Стоимость — от 192 рублей в месяц за пользователя.

Проекты VK WorkSpace — инструмент для управления задачами и проектами в составе платформы VK WorkSpace. Рабочее пространство включает менеджер задач с поддержкой различных представлений: канбан-досок, диаграмм Ганта и таблиц. Сервис поддерживает методологии Agile (Scrum, Kanban) и Waterfall, а также позволяет строить аналитику по задачам, отслеживать прогресс проектов и контролировать загрузку команды. В Проекты можно перенести данные из других систем — например, Jira и Confluence.

В поставку облачной версии бесплатно входит модуль «Страницы» — база знаний для документирования процессов, хранения регламентов, инструкций и результатов встреч. Сервис позволяет руководителю выстроить управление на стратегическом уровне: построить дерево целей, настроить OKR (Objectives and Key Results), каскадировать цели на департаменты и транслировать их в конкретные задачи для исполнителей.

Облачная версия Проектов подключается как дополнительный модуль к основной подписке VK WorkSpace. Это позволяет компаниям заменить набор разрозненных сервисов единой подпиской и работать в экосистеме, где управление проектами интегрировано с корпоративной почтой, мессенджером, видеозвонками и файловым хранилищем. Для быстрого старта в сервисе доступны преднастроенные шаблоны бизнес-процессов для разных отделов, а интерфейс рассчитан на пользователей без опыта работы с системами управления проектами. При необходимости Проекты можно гибко настроить под специфику компании.

Решение доступно по подписке в облаке и в контуре компании.

*«Мы видим растущий спрос на инструменты управления проектами, которые можно подключить без развёртывания инфраструктуры. Облачная версия Проектов VK WorkSpace снижает порог входа: команда получает готовое рабочее пространство с шаблонами процессов, базой знаний и интеграцией с почтой, мессенджером и другими сервисами платформы — от 192 рублей в месяц за пользователя»*, — рассказал **руководитель направления сервисов продуктивности VK Tech Пётр Щеглов**.

**VK Tech** — российский разработчик корпоративного ПО для решения ежедневных задач бизнеса. Портфель VK Tech — это готовая экосистема программных продуктов, которая включает облачную платформу, дата-сервисы, сервисы продуктивности и бизнес-приложения. Продукты VK Tech включены в реестр российского ПО и соответствуют требованиям ФСТЭК, поставляются в формате лицензий (On-Premise) и по подписке (On-Cloud), что делает их доступными для компаний любого масштаба.

**VK WorkSpace** — цифровая среда для совместной работы команд любого масштаба, где сервисы интегрированы между собой и доступны через сквозные сценарии в едином супераппе. Включает в себя корпоративную почту с календарем и адресной книгой, мессенджер, видеоконференции, файловое хранилище со встроенным редактором документов, опросы, заметки, задачи, доску и проекты. Сервисы находятся в едином пространстве, доступны в режиме одного окна и управляются из общей административной панели. У продуктов несколько независимых слоев защиты. Платформу можно развернуть в контуре компании или использовать в облаке. Все сервисы входят в реестр российского ПО и отвечают требованиям 152-ФЗ.
