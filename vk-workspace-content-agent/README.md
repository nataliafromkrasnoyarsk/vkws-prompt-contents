# VK WorkSpace Content Researcher

Агент для поиска тем и форматов текстового контента о VK WorkSpace на основе анализа конкурентов, рынка, трендов и инфоповодов.

## Запуск

```bash
/research-vk-workspace-content
```

Или через активацию:
- "найди темы для контента VK WorkSpace"
- "контент-план VK WorkSpace"
- "анализ конкурентов VK WorkSpace"
- "о чём писать VK WorkSpace"

## Что делает агент

1. **Анализирует контент конкурентов** — Яндекс 360, Битрикс24, МойОфис, Труконф, Kaiten и др.: что публикуют, где, какие форматы, что работает, где пробелы
2. **Исследует рынок** — состояние рынка корпоративных коммуникаций, ключевые изменения, аналитические отчёты
3. **Находит тренды** — AI, импортозамещение, безопасность, суперапп, удалёнка, регуляторика
4. **Мониторит инфоповоды** — продуктовые, рыночные, регуляторные, событийные
5. **Оценивает возможности** — скоринг по модели VISTA (Value, Impact, Standing, Timing, Advantage)
6. **Формирует контент-план** — приоритизированные рекомендации с привязкой к каналам

## Каналы размещения

Агент рекомендует темы для трёх каналов:

| Канал | Форматы | Агент для создания |
|-------|---------|-------------------|
| **Блог workspace.vk.ru** | SEO-статьи: обзоры, гайды, сравнения | `/seo-article-vk-workspace` |
| **VC.ru (VK Tech)** | Кейсы, мнения, тренд-разборы, гайды | `/write-vk-workspace-vc-blog` |
| **Колонки в СМИ** | Экспертные колонки для 14 изданий | `/write-vk-workspace-column` |

## Модель VISTA

Каждая контент-возможность оценивается по 5 критериям:

| Критерий | Вес | Описание |
|----------|-----|----------|
| **V**alue | x2 | Ценность для целевой аудитории |
| **I**mpact | x2 | Потенциальный охват и вовлечение |
| **S**tanding | x1.5 | Усиление позиции VK WorkSpace |
| **T**iming | x1.5 | Актуальность и привязка к моменту |
| **A**dvantage | x1 | Конкурентное преимущество |

Максимум: 85 баллов. Приоритеты: Must-do (70+), Should-do (55-69), Could-do (40-54).

## Результат

Сохраняется в `content-hub/research/drafts/{YYYY-MM-DD}-vk-workspace-content-plan.md`:

- Executive Summary
- Конкурентный анализ
- Рыночные тренды
- Актуальные инфоповоды
- Контент-план по каналам (таблицы)
- Топ-10 детальных рекомендаций
- Календарь публикаций
