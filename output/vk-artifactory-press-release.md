# VK Tech запустила Artifactory — корпоративный менеджер репозиториев для управления артефактами разработки

VK Tech запустила Artifactory — менеджер репозиториев корпоративного уровня для хранения, проксирования и распространения программных артефактов. Решение позволяет компаниям централизованно управлять библиотеками, зависимостями и пакетами, обеспечивая высокую доступность и безопасность данных на территории России. Artifactory доступен как отдельный продукт и в составе платформы VK Dev Platform.

Artifactory выступает единой точкой доступа ко всем программным компонентам: библиотекам, контейнерам, пакетам и другим артефактам, которые используются при сборке приложений. Сервис работает как внутренний репозиторий — разработчики загружают в него собственные компоненты — и одновременно как прокси для внешних источников, таких как Docker Hub или npm registry. Это позволяет контролировать, какие зависимости попадают в продуктовый код, и гарантировать воспроизводимость сборок.

В отличие от open-source решений, Artifactory обеспечивает высокую доступность — систему можно развернуть в отказоустойчивой конфигурации, чтобы сбой одного сервера не останавливал работу команды. Продукт поддерживает девять форматов артефактов: от контейнеров Docker и Helm-чартов для Kubernetes до пакетов npm, PyPi, NuGet, Go, RPM/Yum и RubyGems. Интеграция с объектным хранилищем VK S3 и возможность миграции между хранилищами упрощают внедрение в существующую инфраструктуру.

Artifactory подходит для компаний с разным масштабом и моделью работы. Крупные организации могут развернуть решение на собственных серверах, получив полный контроль над данными и соответствие требованиям регуляторов. Средний бизнес может приобрести продукт как отдельный сервис и быстро запустить работу без затрат на администрирование. Для команд, которые уже используют VK Dev Platform, Artifactory становится частью единой экосистемы с общим окном поддержки и интеграцией с другими сервисами — системой оркестрации контейнеров, мониторингом и объектным хранилищем.

Разработчики работают с Artifactory через привычные инструменты: команды docker pull и docker push, npm install или pip install направляются на внутренний репозиторий вместо публичных источников. Это ускоряет сборку за счёт локального кэширования и снижает риски, связанные с недоступностью внешних сервисов или появлением вредоносных пакетов в публичных репозиториях.

*«Российские компании оказались в ситуации, когда корпоративные инструменты для управления артефактами стали недоступны, а open-source решения не закрывают потребности крупного бизнеса — нет высокой доступности, нет гарантий поддержки, нет соответствия требованиям регуляторов. Artifactory мы разрабатывали для себя — это решение работает в одной из крупнейших ИТ-команд страны. Теперь мы открываем его для рынка, чтобы компании могли получить enterprise-функциональность без зависимости от зарубежных вендоров»*, — комментирует **Дмитрий Лазаренко, руководитель направления облачных и дата-сервисов VK Tech**.

Artifactory может быть развёрнут в инфраструктуре, аттестованной ФСТЭК России, что позволяет использовать его для работы с конфиденциальными данными. Решение соответствует требованиям 152-ФЗ о персональных данных и 187-ФЗ о безопасности критической информационной инфраструктуры. Все данные хранятся на территории Российской Федерации, что исключает риски, связанные с трансграничной передачей информации и санкционными ограничениями.

**VK Tech** — российский разработчик корпоративного ПО для решения ежедневных задач бизнеса. Портфель VK Tech — это готовая экосистема программных продуктов, которая включает облачную платформу, дата-сервисы, сервисы продуктивности, бизнес-приложения. Продукты VK Tech включены в реестр российского ПО и соответствуют требованиям ФСТЭК, поставляются в формате лицензий (On-Premise) и по подписке (On-Cloud), что делает их доступными для компаний любого масштаба.

**VK Cloud** — платформа с широким набором облачных сервисов и ПО для эффективной разработки и работы с данными для компаний любого масштаба. VK Cloud входит в портфель решений VK Tech и базируется на многолетнем опыте развития интернет-сервисов и технологий на базе открытого кода. Среди продуктов VK Cloud инфраструктурные и платформенные облачные сервисы, а также комплексные решения, такие как VK Dev Platform, доступные в Public Cloud, в рамках инсталляции VK Private Cloud и как софт. Клиенты компании — «Битрикс24», «Газпромнефть», «Росатом» и другие крупнейшие игроки своих отраслей.
