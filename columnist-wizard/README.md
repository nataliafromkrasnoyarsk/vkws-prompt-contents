# Columnist Wizard

Агент для создания экспертных колонок и thought leadership контента от лица руководителей VK Tech в крупнейших российских СМИ.

## Возможности

- Написание колонок для **Forbes Russia**, **RBC**, **Vedomosti**, **Kommersant** и других деловых СМИ
- Создание технических статей для **Habr**, **VC.ru**, **CNews**, **TAdviser**
- Адаптация контента под требования конкретного издания
- Соблюдение редакционных стандартов каждой публикации
- Ghostwriting для топ-менеджеров VK Tech

## Типы колонок

1. **Трендовый анализ** — что меняется на рынке и почему это важно
2. **Контринтуитивный взгляд** — вызов общепринятому мнению с доказательствами
3. **Из опыта** — уроки реального проекта, честный разбор
4. **Стратегический комментарий** — реакция на событие или новость
5. **Методология / How-To** — практический фреймворк для читателей

## Использование

### Интерактивный визард
```
/write-column
```
Запустит пошаговый процесс создания колонки с вопросами о:
- Целевом издании
- Типе колонки
- Теме и ключевых тезисах
- Авторе и его экспертизе

### Прямой запрос
```
Напиши колонку про [тема] для [издание] от имени [автор]
```

## Ключевые принципы

### Что делает колонку успешной

1. **Авторитет без высокомерия** — делимся insights, а не лекциями
2. **Российский контекст** — всегда связываем тренды с местным рынком
3. **Практическая ценность** — читатель уходит с чем-то полезным
4. **Ноль маркетинга** — никакого продвижения продуктов
5. **Нативность** — колонка должна чувствоваться написанной ДЛЯ издания

### Критические правила

- **Forbes/RBC/Vedomosti**: категорически запрещена реклама продуктов
- **Habr**: маркетинговая риторика = негатив от сообщества
- **Все издания**: обязательна проверка фактов и источники
- **Деловые СМИ**: формальный деловой стиль обязателен
