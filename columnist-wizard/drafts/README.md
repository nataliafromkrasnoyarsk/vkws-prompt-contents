# Drafts / Черновики колонок

Эта директория используется для сохранения готовых колонок, созданных агентом Columnist Wizard.

## Структура файлов

Файлы именуются по шаблону:
```
[издание]-[дата]-[slug].md
```

Примеры:
- `forbes-2025-01-15-cloud-security.md`
- `rbc-2025-01-20-digital-transformation.md`
- `habr-2025-02-01-kubernetes-best-practices.md`

## Форматы

- `.md` — Markdown файлы (основной формат)
- `.docx` — Word документы (для отправки в редакцию, создаются через pandoc)

## Использование

При финальном согласовании колонки агент предложит варианты сохранения:
1. Файл Markdown — сохраняется сюда
2. Файл DOCX — конвертируется и сохраняется сюда
3. База знаний — добавляется в skills для обучения
4. Комбинация вариантов
