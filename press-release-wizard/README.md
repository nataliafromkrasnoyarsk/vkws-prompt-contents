# Press Release Wizard

Интерактивный агент для написания пресс-релизов в стиле VK Cloud.

## Что делает

Проводит через 10-шаговый процесс создания пресс-релиза: от сбора материалов до финальной валидации. На каждом шаге предлагает варианты и даёт возможность выбора.

## Быстрый старт

```bash
/write-press-release
```

Или быстрый режим (без выбора вариантов):
```bash
/write-press-release --fast
```

## Ключевые возможности

### Исследование рынка (Шаг 3)

Агент может проанализировать рыночный контекст:
- Тренды по теме релиза
- Статистика и прогнозы аналитиков
- Релевантные цифры для усиления текста

### Варианты заголовков (Шаг 5)

Генерируются 3 варианта с разным фокусом:
- Фокус на продукте
- Фокус на ценности
- Фокус на аудитории

### Варианты лидов (Шаг 6)

3 варианта лид-абзаца:
- Классический (Что + Зачем + Контекст)
- С акцентом на проблему
- С цифрой в начале

### Бойлерплейт (Шаг 8)

Агент запрашивает текст бойлерплейта у пользователя (не предлагает готовые варианты).

Запоминает последний использованный бойлерплейт и предлагает его в следующих релизах.

## Команды навигации

| Команда | Действие |
|---------|----------|
| `назад` | Вернуться на предыдущий шаг |
| `статус` | Показать собранные данные |
| `пропустить` | Пропустить опциональный шаг |
| `сначала` | Начать заново |

## Команды доработки (Шаг 10)

| Команда | Действие |
|---------|----------|
| `короче` | Сократить на 20% |
| `подробнее` | Добавить деталей |
| `другой заголовок` | Ещё 3 варианта |
| `другой лид` | Ещё 3 варианта лида |
| `переписать цитату` | Новая версия |
| `добавить цифры` | Усилить фактами |
| `готово` | Завершить работу |

## Экспорт

| Команда | Формат |
|---------|--------|
| `экспорт html` | HTML с SEO-метатегами |
| `экспорт json` | Структурированные данные |
| `экспорт txt` | Plain text для email |

## Валидация

Агент автоматически проверяет:
- Минимум 2 цифры в тексте
- Минимум 1 цифра в цитате
- Отсутствие запрещённых слов
- Наличие бойлерплейта

## Ключевые правила

### Без буллитов в тексте релиза

Текст пишется сплошным текстом, без маркированных списков.

### Расшифровка терминов

Сложные IT-термины объясняются простым языком для журналистов:
```
CI/CD-пайплайн — процесс автоматической сборки и доставки кода
```

### Описание продукта после лида

Структура: Заголовок → Лид → **Описание продукта** → Детали → Цитата → Бойлерплейт

### Бойлерплейт от пользователя

Агент не предлагает готовые варианты, а запрашивает текст у пользователя. Запоминает последний использованный бойлерплейт.

## Стиль VK Cloud

**DO**
- Конкретные цифры: "выросла на 62%"
- Активный залог: "VK Cloud запустила"
- Короткие предложения
- Сплошной текст без буллитов
- Расшифровка терминов

**DON'T**
- Маркированные списки в тексте
- Термины без объяснения
- Превосходные степени: "инновационный"
- Размытые формулировки: "значительно"
- Канцеляризмы: "осуществляет"
